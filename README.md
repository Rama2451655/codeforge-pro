# ⚡ CodeForge Pro — Mobile IDE

A full VS Code experience in your browser. Works on any device.

---

## 🚀 Deploy in 60 Seconds

### Option A — Netlify (No account needed!)
1. Go to **netlify.com/drop** on your phone
2. Upload this ZIP file directly
3. Get a live URL instantly ✅

### Option B — Vercel
1. Push to GitHub (upload files at github.com)
2. Go to **vercel.com** → New Project → Import repo
3. Click Deploy ✅

### Option C — Run Locally
```bash
npm install
npm run dev
# Open http://localhost:5173
```

---

## 📱 Install as Mobile App (PWA)

After deploying:
- **iOS Safari**: Tap Share → "Add to Home Screen"
- **Android Chrome**: Tap Menu → "Install App"

Launches fullscreen like a native app! ✅
